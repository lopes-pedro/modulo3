Teste de imagem de fundo

O tamanho de uma imagem de fundo pode ser definido a partir do atributo CSS background-size e, quando é para ocupar totalmente o tamanho da div, coloca-se os valores como 100% 100%, o primeiro para largura, o segundo para altura. Porém, distorce a imagem

Contudo, isso pode ser resolvido inserindo outros valores para o atributo. A exemplo, o contain, primeiro deles, força a exibição total da imagem, sem distorções, criando barras pretas em partes. O segundo valor, cover, define que o fundo ocupará 100% do tamanho do elemento pai, mesmo que isso corte parte da foto
